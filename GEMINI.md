## 작업 규칙

### 코드 변경 전 필수 (체크 4건)

1. 본 `AGENTS.md` + `design-recheck` + `r1-slice-spine` 읽었는가?
2. 변경하려는 패턴이 매트릭스 §A(이미 채택)에 있다면 추가 결정 없이 따른다
3. 매트릭스 §B(거부)에 있는 것을 도입하려면 **별도 ADR + 사용자 승인** 필수
4. 매트릭스에 없는 패턴 도입 시 매트릭스 §D에 항목 신설 + sub-issue 생성

### 슬라이스 success criterion (R1+ 머지 차단 게이트)

모든 슬라이스 PR은 **다음 4가지 모두 만족** (생략 불가):

1. **새 실행 가능 명령** — `pnpm exec naia-agent ...` 같은 사용자 가치 1줄
2. **단위 테스트 1+** — vitest, 회귀 잡기
3. **통합 검증 1+** — fixture-replay or real-LLM smoke (CI에서 KEY 있을 때만) or 실 backend 호출
4. **README/CHANGELOG entry 1건** — 사용자 향한 변화 기록

(c) 통합 검증 부재 슬라이스 = **머지 거부**.
